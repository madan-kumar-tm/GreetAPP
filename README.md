# 👋 GreetAPP

## 📌 Overview
GreetAPP is a simple web application that displays greeting messages and demonstrates basic frontend development concepts using HTML, CSS, and JavaScript.

---

## 🚀 Features
- Simple and clean UI
- Displays greeting messages
- Lightweight and fast
- Easy to customize

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 📁 Project Structure
GreetAPP/
│── index.html
│── style.css
│── script.js
│── README.md

---

## ▶️ How to Run

1. Clone the repository:
git clone https://github.com/madan-kumar-tm/GreetAPP.git

2. Navigate to the project folder:
cd GreetAPP

3. Open `index.html` in your browser.

---

## 🎯 Purpose
This project was built to practice:
- Frontend fundamentals
- DOM manipulation
- Basic project structure

---

## 🔧 Customization
You can update:
- Greeting logic in `script.js`
- Styles in `style.css`
- Layout in `index.html`

## 📈 Future Improvements
